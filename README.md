C-Raycaster
===========
This is just a random project I decided to make because I remember [TheCodingTrain](https://www.youtube.com/@TheCodingTrain) doing it, and I thought this was a cool idea.

CMake Dependencies
==================
Raylib v2.28.0

Build Instructions
==================
```console
$ mkdir build
$ cd build
$ cmake .. -G Ninja
$ ninja
$ ./c-raycaster
```
